# Workflow : Bilan quotidien multi-plateformes

## Objectif
Produire chaque matin un bilan unique regroupant les performances des réseaux suivants :
- LinkedIn  
- Instagram / TikTok  
- YouTube (Shorts + Longs)  
- WhatsApp Channel  
- Podcasts (Happy Work + Happy Moi)

Ce bilan doit permettre à Gaël de prendre rapidement 3 à 5 décisions utiles pour la journée (contenus à pousser, horaires, recyclage, signaux faibles, opportunités).

---

## Étape 1 — Appel des agents individuels
Chaque matin, le workflow exécute les agents suivants :

1. **Agent LinkedIn**  
   → renvoie : meilleur post, engagement, croissance, idée de post du jour.

2. **Agent Instagram/TikTok**  
   → renvoie : meilleur Reel, taux de complétion, % non-followers, idée de Reel du jour.

3. **Agent YouTube**  
   → renvoie : Short N°1, watchtime vidéo longue, taux de rétention, recommandations.

4. **Agent WhatsApp**  
   → renvoie : croissance, sondage le plus puissant, tendance à réutiliser.

5. **Agent Podcasts**  
   → renvoie : top épisodes, plateformes actives, recommandation de sujet.

Chaque agent produit un mini-rapport structuré (texte normal).

---

## Étape 2 — Consolidation par l’Agent Analytics Global
L’Agent Analytics Global récupère tous les mini-rapports ci-dessus et génère :

### 1. Résumé général (5 lignes)
- le réseau qui monte le plus
- le réseau qui ralentit
- le contenu global N°1 des dernières 24h
- une anomalie éventuelle (baisse forte)
- une opportunité immédiate

### 2. Analyse détaillée par réseau (1 bloc par plateforme)
Pour chaque plateforme :
- performances des dernières 24h
- meilleur contenu + raison du succès
- chute éventuelle + hypothèse
- tendance à surveiller

### 3. Top 3 contenus toutes plateformes confondues
- (1) contenu le plus performant  
- (2) contenu à potentiel de recyclage  
- (3) contenu sous-estimé à booster

### 4. Plan d’action du jour (3 actions maximum)
Exemples :
- republier un extrait en Short  
- recadrer un Reel en version punchline  
- booster un thème sur LinkedIn  
- pousser un épisode Happy Work  
- mettre un sondage WhatsApp pour valider un angle  

---

## Étape 3 — Format final du rapport
Le rapport doit être généré dans cet ordre :

1. **Résumé global**  
2. **Analyse LinkedIn**  
3. **Analyse Instagram/TikTok**  
4. **Analyse YouTube**  
5. **Analyse WhatsApp**  
6. **Analyse Podcasts**  
7. **Top 3 contenus du jour**  
8. **Actions recommandées**  

---

## Étape 4 — Fréquence
Le bilan est produit **chaque matin**, idéalement avant 10h.

---

## Étape 5 — Utilisation
Ce bilan servira à :
- choisir les contenus prioritaires,
- décider quel épisode mettre en avant,
- ajuster les heures de publication,
- repérer ce qui fonctionne auprès de ta communauté,
- construire un cercle vertueux d’amélioration quotidienne.
